## 📂 Estrutura do Projeto

A estrutura básica do projeto é organizada da seguinte forma:

```
cypress/
│
├── e2e/               # Pasta onde ficam os testes end-to-end.
│
├── fixtures/          # Contém arquivos de dados mockados (exemplo: example.json).
│
├── support/           # Scripts de suporte (ex.: comandos customizados).
│
cypress.config.js      # Arquivo de configuração principal do Cypress.
```

---

## 🚀 Como clonar e configurar o projeto

1. **Clone o repositório em sua máquina local:**

   ```bash
   git clone https://github.com/alessandraasantos/projeto-avanti.git
   ```

2. **Acesse a pasta do projeto:**

   ```bash
   cd projeto-avanti
   ```

3. **Configure o repositório remoto (caso ainda não esteja configurado):**

   ```bash
   git remote add origin https://github.com/alessandraasantos/projeto-avanti.git
   ```

---

## ✔️ Requisitos

- Node.js instalado
- Cypress instalado (ou instale usando `npm install` se houver um `package.json`)

---

## ✅ Como executar os testes

Após clonar o projeto, execute:

```bash
npx cypress open
```

Ou para rodar em modo headless:

```bash
npx cypress run
```
